#  Bitcoin Market Sentiment vs Trader Performance Analysis

##  Overview
This project explores the relationship between Bitcoin market sentiment (Fear, Greed, Neutral, Extreme Greed) and trader performance using historical trading data from Hyperliquid.

The goal is to understand how market psychology impacts profitability, risk, and trading behavior.

---

##  Dataset
- Bitcoin Fear & Greed Index (daily sentiment data)
- Hyperliquid historical trading data (trader-level transactions)

---

##  Workflow
1. Data cleaning and preprocessing
2. Timestamp standardization
3. Merging sentiment with trading data
4. Feature engineering (win rate, risk, trade size)
5. Exploratory data analysis
6. Sentiment-based performance analysis
7. Visualization of key insights

---

##  Key Insights
- Highest profitability occurs during **Greed** phases
- Win rate increases with positive sentiment
- Trade size increases during extreme sentiment conditions
- Market sentiment strongly influences trader behavior

---

##  Conclusion
Market sentiment is a strong predictor of trading behavior and performance. Traders tend to take higher risks during extreme market emotions, which impacts profitability patterns.
